# C Quick Reference

## Format Specifiers
%d - integer
%f - float (%.2f untuk 2 desimal)
%c - character
%s - string
%p - pointer address

## Basic Structure
#include <stdio.h>
int main() {
    // code here
    return 0;
}

## Common Functions
printf("text %d", var);   // Output
scanf("%d", &var);        // Input
sizeof(var);              // Ukuran variabel
